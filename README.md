# Installation

- Clone the repo and goto the dir
```sh
$ git clone https://github.com/r3yc0n1c/flipkart-clone.git
$ cd flipkart-clone
```
- Install dependencies
```sh
$ cd frontend
$ npm i .
$ cd ../backend
$ npm i .
```
- Start the server
```sh
$ cd backend
$ npm start
```
- Start the client
```sh
$ cd frontend
$ npm start
```
