# Flipkart Clone
Clone of https://www.flipkart.com/

# Built With

- **M**ongoDB
- **E**xpress
- **R**eact
- **N**ode

# Installation

- Clone the repo and goto the dir
```sh
$ git clone https://github.com/r3yc0n1c/flipkart-clone.git
$ cd flipkart-clone
```
- Install dependencies
```sh
$ npm i .
```
- Start the server
```sh
$ npm start
```
